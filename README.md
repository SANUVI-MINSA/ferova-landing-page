# Ferova Landing Page

## ◈ Descripción

Landing page oficial de **Ferova**, una plataforma digital enfocada en el seguimiento del tratamiento de anemia materno-infantil en el Perú.

El proyecto busca conectar a madres, cuidadores y personal de salud del MINSA mediante una experiencia moderna, accesible y centrada en el acompañamiento del paciente.

---

## ⌘ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Google Fonts
- Lucide Icons
